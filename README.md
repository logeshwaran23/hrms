# HRMS Portal

A React + Node.js implementation for the HRMS Employee Self-Service Portal.

## Project Structure

- `frontend` — React UI built with Vite
- `backend` — Express API for authentication, attendance, leave, and daily summaries

## Development

1. Open terminal in project root.
2. Run `npm install` to install root helper dependencies.
3. Run `npm run start` to launch both frontend and backend together.

Alternatively run separately:
- `npm run frontend`
- `npm run backend`
